TRAINING CURRICULUM

WEB APPLICATION DEVELOPMENT

The application is built around book library theme, accessible through browser and running on a web server. The backend is written in Java EE and the frontend on JSF and PrimeFaces frameworks to create a user-friendly interface. The MySQL database is used to store information about books and users, allowing for efficient management of the library.

APPLICATION SERVERS ADMINISTRATION & DEPLOYMENT

Linux CentOS9 & Windows 10 Enterprise system administration, user management, monitoring and troubleshooting application servers, software installation and configuration, security management, scripting and automation. Application servers used are WildFly 26 and WebSphere 8.5.5 as a showcase.

SECURITY & CRYPTOGRAPHIC PROTOCOLS

The application is configured to use SSL/TLS protocol with self-signed root certificate as a proof of concept. On WildFly application server, HTTP to HTTPS redirection is also configured. WebSphere application server was installed on remote virtual machine and Putty tool was used for file transfer - configured on SSH - Secure Shell Protocol.

BUILD AUTOMATION

Maven was used as a tool for building, packaging and managing project dependencies. As a showcase, the application was packaged with bundled external libraries and without external libraries which in this case was provided by configured shared libraries on application server.

SOFTWARE ARCHITECTURE

The application is packaged as multi-module architecture. Presentation layer is represented by web module (.war), business layer is represented as ejb module(.ejb) and persistence layer is packaged as JPA module(.jar). All modules are bundled into EAR module into .ear archive.

SCRIPTING & TESTING

Groovy scripting language was used as proof of concept for testing REST endpoints. Also Postman and Insomnia tools were used for testing and documenting HTTP APIs.
